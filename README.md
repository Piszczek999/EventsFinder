## Overview

EventFinder is a portfolio web application built with **Next.js** that allows users to discover, create, and manage local events.

Users can browse events, filter them by category, search by location, and register for events they are interested in. The system is designed to provide a simple and intuitive way to explore activities happening nearby, whether at home or while traveling.

---

## Features

- Browse available events
- Search events by location
- Filter and sort events by category
- View detailed event pages
- Create and publish new events
- Register for events
- Manage event participation
- Responsive UI for mobile and desktop
